cara memakai website ini adalah dengan cukup login melalui username dan password dari admin

setelah itu kalian akan mendapatkan fitur berbeda setiap role

role admin = bisa mengedit dan menambah data siswa maupun guru
role guru = bisa mengabsensi dan melihat data siswa setiap kelas yang mereka ajar
role siswa = hanya bisa untuk absen dan scan QR
